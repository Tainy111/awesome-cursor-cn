## 🎯 关于 Cursor

Cursor 是 AI 原生代码编辑器，基于 VS Code 构建，内置 GPT-4、Claude 等大模型，让编程效率提升 10 倍。

## 💡 进阶技巧

### 多文件编辑
```
@file:src/main.js @file:src/utils.js
请重构这两个文件，提取公共逻辑
```

### 上下文感知
```
@code:当前选中的代码
请优化这段代码的性能
```

### 图像理解
- 直接粘贴 UI 设计图，让 AI 生成代码
- 截图错误信息，自动诊断问题
